How to create a Discord webhook

  1. Open your Discord server and click the server name → Server Settings.
  2. Go to Integrations → Webhooks.
  3. Click New Webhook.
  4. Choose the channel where updates should be posted.
  5. Copy the Webhook URL.
  6. Paste the URL below and save the subscription.

Tip: You can rename the webhook and set a custom avatar in Discord.
